# turakhia-dis-2021

In this paper, the authors develop tools for designers to help them design adaptive learning solutions (for motor skills training).

They conduct on a formative study, namely giving a design task to their students, so that they identify what is challenging for designers when designing adaptive motor skill tools. Then, they implement the tool to support them.
